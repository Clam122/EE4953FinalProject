# Key Distribution Server Proof of Concept.

This project is a demo of Distributed PGP key servers. It is not intended for production use, has not been audited by any cybersecurity organization or professional. In all likelyhood, it is horribly insecure as a service. 

A client.py script is provided to interface with the program over the terminal.
